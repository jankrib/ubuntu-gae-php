# ubuntu-gae-php
Template for Google App Engine PHP with vagrant

1. Clone the repo
> $ git clone ...

2. Start the vagrant box
> $ vagrant up

3. Browse to http://127.0.0.1:8080


Admin is available at:
http://127.0.0.1:8000


PhpMyAdmin is available at:
http://127.0.0.1:8888
User name: root
Password: vagrant
